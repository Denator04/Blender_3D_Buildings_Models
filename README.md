# Blender_3D_Buildings_Models
Modele 3D budynków znajdujących się na obszarze kampusu mojego uniwersytetu - Akademia Górniczo Hutnicza w Krakowie (AGH University)

Na potrzeby projektu w moim kółku naukowym wykonałem modele 3D w programie Blender budynków:
- A0 - główny budynek
- Bratek - jeden z akademików znajdujący się na Miasteczku Studenckim AGH
- Klub Studio - budynek leżący nieopodal Miasteczka Studenckiego
